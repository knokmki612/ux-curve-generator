How to use UX Curve Generator

What is UX Curve?

UX Curve is one of the UX(User Experience) evaluation methods. This method has the advantage to able to evaluating long-term UX that due to express as a curve line that a transition of user experience by a thing or a service.

Change points of UX Curve in UX Curve Generator

UX Curve Generator modifies the detail from UX Curve. The changes were impacted from UX Graph which is a variant of UX Curve.

  • Uses only a score of satisfaction as a subjective index
  • Put the evaluation of before use in addition to an episode

Introduction

  1. Input a title
  2. Adjust a score of "Expected" and "At Present"
  3. You can look at the transition the slope of the straight line
  4. If could make the slope of the line as you imagine, click an "Add" button to create a new "Episode"
  5. You can look at the straight line became a curve with a new number of coordinate points after creating an episode
  6. Changing a score and date moves coordinate point
  7. You can look at the curve became complexity after creating more episodes
  8. Cross the order of coordinate points in the Horizontal axis also change the order of episodes list because the episodes list sorted by date
  9. If could make the curve as you imagine, input a description so that match the shape of curve
  10. Hooray, you made a UX curve! Already generated the embed code so you can place your UX Curve in your blog etc.

Actually, you can input a title, date, score, and description regardless of order as you like except when creating a new episode.

Overview of User Interfaces

Title

Subject of the drawing UX curve.

UX Curve

Drawn UX Curve.

When updated "Episode" or "Expected" or "At Present", UX Curve will automatically redraw.

Episode

Put the event or the impression relates to the subject of drawing UX Curve.

Can input multiple episodes.

Episodes are automatically sorted by date.

Items describe as follows.

Date

Specify when the episode occurred.

Can put relatively from previous/next episode or put a strict date.

Score of Satisfaction

Put the score of satisfaction based on occurred event or impression.

0 means neutral, 100 means most satisfied, -100 means most unsatisfied.

Description

Put the description of occurred event or impression.

Expected

Put the impression of before use.

items are the same as "Episode" except no date.

At Present

Put the at present impression.

items are the same as "Episode" except no date.

Add Button

When click the button will show a component which creates a new episode.

The button is placed multiple between episodes.

The above and below episode behave as previous/next episode during input a relative date

Embed Code

Displays the widget code that saves and views the UX Curve. It can embed to HTML.
